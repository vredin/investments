## Auth

Session-based via `starlette.middleware.sessions` (signed cookie, `SESSION_MAX_AGE` from config). `login_required` dependency injected on protected routers. No OAuth, no JWT.

## LLM Integration

OpenRouter (openai-compatible endpoint). Used in `services/llm.py`. No Anthropic SDK — see `.ccs/memory` for rationale. Embeddings via `sentence-transformers` (1024-dim), stored in pgvector.
