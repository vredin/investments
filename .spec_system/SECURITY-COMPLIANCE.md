## Regulatory Compliance Status

### Personal Use Declaration

This system is built and operated for sole personal use by a single individual. It does not provide investment advice to third parties. All outputs include the disclaimer: "Not investment advice. For informational purposes only."

**Applicable framework**: Personal use exemption under MiFID II (Directive 2014/65/EU) and Ukrainian Law on Capital Markets (No. 3480-IV, 2021). These exemptions apply only as long as:
1. The system is used exclusively by the builder/owner
2. Outputs are never shared with third parties
3. The system has no trading execution capability (read-only CSV import only)

**If any of these conditions change**, legal review is required before continuing operation.
